# Akadeni
## Overview

Akadeni is a free android application dedicated to merchant who aim to
track LOANS on their go. It allows recording loan with information such as
- full name
- amount
- date due

Merchant will also be able to mark loan as partially paid or totally paid by 
just entering pad amount.
